What is CSS?

cascading style sheets (CSS) is used to format the layout of a webpage.

With CSS you can control the color,font, the size of text, the spacing between elements,how elements are positioned and laid out,What background images or background colors are to be used, diferent displays for diferent devices and screen size,and much more!
